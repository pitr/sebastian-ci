Sebastian CI

Open source continuous integration tool with tight integration to GitHub.

sebastian.war worker box View on GitHub

Goals

Sebastian CI is a replacement for other CI tools, learning from previous solutions:

  1. Integration with GitHub (e.g. Pull Request-awareness)
  2. Deploying should be as simple as java -jar sebastian.war
  3. Isolate build environments using Vagrant (Virtual Box)
